# Infinite Minesweeper

Very unoptimized, written as a test to try out raylib-python

## Installation

1. Clone this repo
```
git clone https://github.com/rfblock/infinite-sweeper.git
cd infinite-sweeper
```
2. Create python virtual environment
```
python -m venv venv
pip install -r requirements.txt
```
3. Run
```
python main.py
```
